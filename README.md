# Wieck Media Scala Project Template

Clone this repo then run `g8` pointing to the file path of the repo.

    g8 file://path/to/repo

You now have a spiffy new Scala/SBT project.

    sbt
		> test

## Features

* A starter `build.sbt`
* `Spec` trait that extends `FreeSpec`

## giter8 Installation

    brew install giter8
